# Windows Telemetry OFF

Интерактивный Batch-скрипт для отключения телеметрии, диагностических задач и повышения конфиденциальности в операционных системах Windows 10 и 11. Скрипт не удаляет системные компоненты и предлагает несколько уровней настройки, включая возможность восстановления стандартных параметров.

## Описание

Скрипт предназначен для снижения фоновой активности операционной системы, связанной со сбором данных, историей активности, рекомендациями и обратной связью. Все изменения применяются через штатные механизмы Windows: реестр (политики), управление службами и планировщик задач.

Ключевое отличие скрипта — безопасность. Утилита не нарушает целостность системы, не затрагивает критически важные компоненты (Центр обновления Windows, Microsoft Store, Windows Defender) и позволяет вернуть первоначальные настройки.

## Уровни применения (Режимы)

Утилита предоставляет три режима работы в зависимости от требуемой степени вмешательства:

1. **Safe** (Безопасный)
   - Отключение истории активности и рекламного идентификатора.
   - Отключение Consumer Features.
   - Ограничение сбора геолокации.
   - Отключение веб-поиска в меню Пуск.
   - Блокировка рекомендаций, подсказок и запросов обратной связи.

2. **Balanced** (Сбалансированный)
   - Включает все изменения режима **Safe**.
   - Отключение основных служб телеметрии (`DiagTrack` и `dmwappushservice`).
   - Отключение базовых диагностических задач (CEIP, Compatibility Appraiser).

3. **Pro** (Продвинутый)
   - Включает все изменения режима **Balanced**.
   - Глубокое отключение диагностических задач, задач оценки энергопотребления и синхронизации.
   - Жесткое ограничение персонализации, отправки образцов в Microsoft MAPS и отключение Windows Spotlight.

## Дополнительные функции

- **Проверка текущего состояния**: Позволяет быстро проанализировать систему и узнать статус основных политик конфиденциальности, служб и задач телеметрии.
- **Восстановление стандартных настроек**: Интегрированная функция отката, которая снимает примененные ограничения, возвращает службы в ручной режим запуска и активирует отключенные задачи.

## Использование

1. Скачайте последнюю версию скрипта со страницы Releases на GitHub.
2. Запустите файл `Windows-Telemetry-OFF.bat`.
   *Примечание: Скрипт автоматически запросит права администратора, которые необходимы для внесения изменений в системный реестр и управления службами.*
3. В появившемся интерактивном меню выберите требуемый режим работы путем ввода соответствующей цифры.
4. После применения изменений рекомендуется перезагрузить систему.

Результаты выполнения сохраняются в директории со скриптом в файлах отчетов `Windows_Telemetry_OFF_Log.txt` и `Windows_Telemetry_OFF_Result.txt`.

## Системные требования

- **ОС**: Windows 10 / Windows 11 (любые редакции).
- **Права**: Требуются права локального администратора.

## Важное уведомление

Внесение изменений в системные параметры производится под вашу ответственность. Обратите внимание, что крупные системные обновления Windows могут сбрасывать некоторые измененные настройки на значения по умолчанию. Для контроля параметров рекомендуется периодически использовать функцию проверки текущего состояния в меню утилиты.
